
AI Tutor SaaS - Knowledge Base & Architecture Guide
==================================================

VISION
------
Build a multi-tenant SaaS platform for schools where:
- Teachers upload PDFs, notes, PPTs, worksheets, assignments, question papers.
- Students access learning materials.
- Students take quizzes, tests, assignments, tryouts.
- System tracks mastery at topic level.
- AI detects knowledge gaps.
- AI generates bridge learning paths.
- AI generates personalized lessons, worksheets, quizzes, and study plans.

CORE PRINCIPLE
--------------
The LLM is NOT the intelligence layer.

The intelligence layer consists of:
1. Curriculum Graph
2. Assessment Engine
3. Mastery Engine
4. Error Classification Engine
5. Gap Detection Engine
6. Learning Path Engine
7. Recommendation Engine

The LLM generates content based on these outputs.

MASTERY ENGINE
--------------

For each student:

{
  "studentId": "123",
  "topic": "LCM",
  "mastery": 42
}

Mastery Formula:

Accuracy
+ Attempts
+ Time Taken
+ Recency

Initially:

mastery = correct_answers / total_answers

Calculation Error:

Reading Error:
- Misread question

Logic Error:
